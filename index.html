Guía Rápida de Atajos de Teclado

YouTube

  • K / Espacio: Reproducir / Pausar video
  • J: Retroceder 10 segundos
  • L: Avanzar 10 segundos
  • M: Silenciar / Activar sonido
  • F: Activar / Desactivar pantalla completa
  • T: Activar / Desactivar modo cine
  • I: Abrir el Mini reproductor
  • Esc: Cerrar Mini reproductor o pantalla completa
  • Flecha Arriba / Flecha Abajo: Subir / Bajar volumen
  • Flecha Izquierda / Flecha Derecha: Retroceder / Avanzar 5 segundos (en la barra de progreso)
  • C: Activar / Desactivar subtítulos (si están disponibles)
  • Shift + N: Siguiente video (en una lista de reproducción)
  • Shift + P: Video anterior (en una lista de reproducción)
  • Números 1-9: Saltar al 10%-90% del video
  • Número 0: Saltar al inicio del video

Visual Studio Code (VS Code)

(Nota: Cmd en macOS equivale a Ctrl en Windows/Linux en muchos casos. Se usa Ctrl como referencia general)

  • Ctrl + S: Guardar archivo actual
  • Ctrl + Shift + S: Guardar como...
  • Ctrl + N: Nuevo archivo
  • Ctrl + O: Abrir archivo
  • Ctrl + W: Cerrar archivo actual
  • Ctrl + K Ctrl + W: Cerrar todos los archivos
  • Ctrl + C: Copiar selección
  • Ctrl + X: Cortar selección
  • Ctrl + V: Pegar
  • Ctrl + Z: Deshacer
  • Ctrl + Y / Ctrl + Shift + Z: Rehacer
  • Ctrl + F: Buscar en el archivo actual
  • Ctrl + H: Reemplazar en el archivo actual
  • Ctrl + Shift + F: Buscar en todos los archivos del proyecto
  • Ctrl + P: Ir a archivo...
  • Ctrl + Shift + P / F1: Mostrar Paleta de Comandos
  • Ctrl + `: Abrir / Cerrar Terminal Integrada
  • Alt + Flecha Arriba / Flecha Abajo: Mover línea arriba / abajo
  • Shift + Alt + Flecha Arriba / Flecha Abajo: Copiar línea arriba / abajo
  • Ctrl + /: Comentar / Descomentar línea actual o selección
  • Ctrl + B: Mostrar / Ocultar la barra lateral

Google Chrome

  • Ctrl + T: Abrir nueva pestaña
  • Ctrl + W / Ctrl + F4: Cerrar pestaña actual
  • Ctrl + Shift + T: Reabrir última pestaña cerrada
  • Ctrl + Tab / Ctrl + PgDn: Ir a la siguiente pestaña
  • Ctrl + Shift + Tab / Ctrl + PgUp: Ir a la pestaña anterior
  • Ctrl + L / Alt + D: Ir a la barra de direcciones
  • F5 / Ctrl + R: Recargar página
  • Ctrl + H: Abrir historial
  • Ctrl + J: Abrir descargas

Gmail

(Nota: Es posible que necesites activar los atajos de teclado en la configuración de Gmail)

  • C: Redactar nuevo correo
  • /: Buscar en Gmail
  • R: Responder
  • A: Responder a todos
  • F: Reenviar
  • E: Archivar conversación
  • #: Mover a la papelera
  • L: Abrir menú "Etiquetar como"
  • G + I: Ir a Recibidos
  • G + S: Ir a Destacados
  • G + T: Ir a Enviados
  • Ctrl + Enter: Enviar correo

Google Docs

  • Ctrl + B: Negrita
  • Ctrl + I: Cursiva
  • Ctrl + U: Subrayado
  • Ctrl + K: Insertar enlace
  • Ctrl + Alt + M: Insertar comentario
  • Ctrl + Shift + C: Contar palabras
  • Ctrl + Shift + L: Alinear a la izquierda
  • Ctrl + Shift + E: Centrar
  • Ctrl + Shift + R: Alinear a la derecha
  • Ctrl + Shift + 7: Lista numerada
  • Ctrl + Shift + 8: Lista con viñetas

Slack

  • Ctrl + K / Cmd + K: Abrir buscador rápido (canales, mensajes, personas)
  • Alt + Flecha Arriba/Abajo: Navegar entre canales/mensajes directos
  • Esc: Marcar todos los mensajes como leídos
  • Shift + Esc: Marcar mensajes del canal actual como leídos
  • Ctrl + F / Cmd + F: Buscar en el canal o mensaje actual
  • Ctrl + Shift + M / Cmd + Shift + M: Ver Actividad
  • Ctrl + . / Cmd + .: Mostrar/Ocultar panel derecho (detalles, hilos)
  • E: Editar último mensaje enviado

Trello

  • B: Abrir menú de tableros
  • /: Enfocar campo de búsqueda
  • N: Añadir nueva tarjeta (estando sobre una lista)
  • F: Abrir filtro de tarjetas
  • Q: Mostrar solo tarjetas asignadas a mí
  • Espacio: Asignarme/desasignarme de una tarjeta (estando sobre ella)
  • M: Añadir/quitar miembros (estando sobre una tarjeta)
  • L: Abrir menú de etiquetas (estando sobre una tarjeta)
  • Esc: Cerrar diálogo o menú

Figma

  • V: Herramienta Mover (Seleccionar)
  • F: Herramienta Frame
  • R: Herramienta Rectángulo
  • O: Herramienta Elipse
  • L: Herramienta Línea
  • T: Herramienta Texto
  • Ctrl + G / Cmd + G: Agrupar selección
  • Ctrl + Shift + G / Cmd + Shift + G: Desagrupar
  • Ctrl + D / Cmd + D: Duplicar
  • Alt + Arrastrar: Duplicar al arrastrar
  • Ctrl + / / Cmd + /: Buscar menús y comandos

GitHub

(Atajos para usar dentro de github.com)

  • S / /: Enfocar barra de búsqueda
  • G + N: Ir a Notificaciones
  • G + D: Ir al Dashboard (Feed)
  • G + C: Ir al código (en un repositorio)
  • G + I: Ir a Issues (en un repositorio)
  • G + P: Ir a Pull Requests (en un repositorio)
  • T: Activar buscador de archivos (en un repositorio)
  • W: Cambiar de rama o etiqueta (en un repositorio)
  • ?: Mostrar panel de atajos de teclado
